# Lottery Game

## Installation

requires [NVM](https://github.com/nvm-sh/nvm)
```sh
nvm install v.12.13.0
nvm use v.12.13.0
```
or requires [Node.js](https://nodejs.org/) v12+ to run.

## Development
```sh
npm install
node run dev
```

## production
```
npm start
```

app run on `http://localhost:9000`
